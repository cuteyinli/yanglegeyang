# Forest 🌲

一款基于微信小游戏平台开发的「羊了个羊」风格消除类小游戏。

## 项目结构

```
├── client/          # 微信小游戏前端
│   ├── audio/       # 音频资源
│   ├── images/      # 图像资源
│   ├── scenes/      # 游戏场景
│   ├── game.js      # 入口文件
│   ├── game.json    # 运行环境配置
│   └── global.js    # 全局状态管理
├── server/          # 后端服务
└── docs/            # 项目文档
    ├── 前端/
    ├── 后端/
    └── 数据库/
```

## 技术栈

- **前端**：微信小游戏原生 Canvas API
- **后端**：待定
- **数据库**：待定

## 快速开始

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开微信开发者工具，导入 `client` 目录作为项目
3. 即可在模拟器中预览运行
