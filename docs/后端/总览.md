# 后端总览

## 技术选型

- **语言**：Python 3.10+
- **框架**：FastAPI
- **异步支持**：uvicorn（ASGI 服务器）
- **ORM**：SQLAlchemy 2.0（异步模式）
- **鉴权**：微信 OpenID + JWT Token

## API 接口设计

### 1. 登录鉴权

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/auth/login` | 微信登录（code 换 token） |

### 2. 关卡管理

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/level/config` | 获取当前关卡配置 |
| GET | `/api/level/daily` | 获取每日挑战关卡 |

### 3. 游戏记录

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/game/start` | 开始游戏（记录开始时间） |
| POST | `/api/game/result` | 上报游戏结果 |

### 4. 排行榜

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/rank/top` | 获取排行榜 Top N |
| GET | `/api/rank/me` | 获取当前用户排名 |

## 核心流程

### 登录流程

```
小游戏端 wx.login() 获取 code
       ↓
POST /api/auth/login { code }
       ↓
后端用 code 调用微信接口换取 openid + session_key
       ↓
查找/创建用户，生成 JWT Token 返回
       ↓
小游戏端存储 Token，后续请求携带 Authorization 头
```

### 游戏流程

```
请求关卡配置 → 渲染游戏 → 玩家操作 → 游戏结束 → 上报结果 → 更新排行榜
```

## 部署方案

- **开发环境**：`uvicorn main:app --reload --port 8000`
- **生产环境**：Docker + Nginx 反向代理 + Gunicorn(w/ uvicorn workers)
